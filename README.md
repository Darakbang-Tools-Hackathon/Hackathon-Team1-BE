# VITA Track: AI 기반 노인 일상 활동 모니터링 시스템

VITA Track은 노인의 가정 내 일상 활동 소리를 분류하고 데이터를 분석하여, 보호자에게 직관적인 건강 상태 모니터링 및 이상 징후 알림을 제공하는 헬스케어 시스템입니다.

---

## 🎯 문제 정의 (Problem Definition)

노화에 따라 건강이 점진적으로 저하되는 것은 자연스러운 현상입니다. 하지만 건강이 급격히 악화되는 시기에 적절히 대응하지 않으면, 영구적인 건강 손상으로 이어질 수 있습니다.

이는 노인 개인의 건강 문제를 넘어, 가족의 돌봄 부담과 의료비 등 사회적 비용(social cost)을 가중시키는 요인이 됩니다.

## 🚀 핵심 솔루션 (Core Solution)

VITA Track은 노인의 가정에 설치되어 일상 활동 소리를 분류하는 시스템입니다. 이 시스템은 분석된 데이터를 기반으로 노인의 건강 상태를 **'건강 지수(Health Score)'**로 점수화하여 지속적으로 모니터링합니다.

만약 평소의 일상 패턴과 다른 변화(anomaly)가 감지되어 이 건강 점수가 하락하면, 시스템은 즉시 보호자에게 알림을 전송하여 선제적인 대응을 지원합니다.

## 주요 기능 (Features)

### 1. 데이터 기반 생활 건강 모니터링 대시보드

설치된 IoT 센서(활동 감지, 수면 패턴, 실내 환경 등)로부터 수집된 시계열 데이터(time-series data)를 실시간으로 분석합니다. 이 데이터를 종합하여 '생활 건강 지수(Health Score)'를 산출하고, 보호자가 웹 대시보드(Web Dashboard)를 통해 직관적인 그래프 형태로 노인의 건강 상태 변화 추이를 모니터링할 수 있도록 지원합니다.

* **직관적 상태 파악:** 복잡한 로우 데이터(raw data)가 아닌, 점수화 및 시각화된 건강 지수를 제공하여 보호자가 노인의 상태를 즉각적으로 파악할 수 있습니다.
* **이상 징후 조기 감지:** 장기적인 건강 트렌드 분석을 통해 평소와 다른 패턴이나 건강 지수의 급격한 하락 등 이상 징후를 조기에 감지합니다.
* **선제적 대응:** 이상 징후 감지 시, 보호자에게 즉각적인 푸시 알림(push notification)을 전송하여 신속한 확인 및 선제적 대응을 가능하게 합니다.

### 2. AI 기반 최적 센서 설치 위치 추천

사용자가 주거 공간의 도면을 업로드하면, `Gemini API`와 같은 멀티모달(multi-modal) AI 모델이 이미지 분석(image analysis)을 수행합니다. AI는 도면상의 공간 구조(침실, 화장실 등)와 주요 생활 동선을 식별합니다.

이 분석 결과를 바탕으로, 가장 효율적인 데이터 수집 및 위험 감지가 가능한 최적의 센서 설치 위치를 도면 위에 시각적으로 제안합니다.

## 기술 스택 (Tech Stack) 

* **Backend:** Java (Spring Boot)
* **Frontend:** React Vite, Tailwind CSS
* **Database:** InfluxDB (시계열 데이터), MySQL (사용자 정보)
